常用于主动操作后的反馈提示。与 Notification 的区别是后者更多用于系统级通知的被动提醒。 Vmd 注册了一个LsMessage方法用于调用，可以接收一个字符串作为参数，它会被显示为正文内容。 在使用html开发中使用的是Vmd.Message("这是一条消息提示")。 在main.js里完全引入，用this.$message('这是一条消息提示')。







Options 参数 说明 类型 可选值 默认值 message 消息文字 string — — type 主题 string success/warning/info/error info iconClass 自定义图标的类名，会覆盖 type string — — customClass 自定义类名 string — — duration 显示时间, 毫秒。设为 0 则不会自动关闭 number — 3000 showClose 是否显示关闭按钮 boolean — false onClose 关闭时的回调函数, 参数为被关闭的 message 实例 function — — Message 方法 方法名 说明 close 关闭当前的 Message
